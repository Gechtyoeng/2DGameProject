# 2D Escape Bird Game

## Short Introduction
This is a 2D Flappy Bird-style game built with **Unity**. The player controls a bird that must navigate through obstacles without hitting them.

## Game Objective
- The main goal is to achieve the highest score by passing through pipes.
- You **win the game** when the score reaches **10**.

## Game Rules
1. Press the **Start** button to begin.
2. Tap or click to make the bird fly.
3. Avoid hitting the pipes or the ground.
4. The game ends if the bird hits an obstacle.
5. Reach a score of **10** to win.
